# AI Collaboration Workflow

本项目把 AI 协作做成普通文档和 prompt 模板，不把本机 agent skill 体系作为运行依赖。

## Recommended Loop

1. 明确本轮目标：导出、格式检查、章节润色、引用核对或交付复核。
2. 准备最小材料：只提供必要章节、规则、错误报告或检查清单。
3. 使用 `prompts/` 中的任务型模板。
4. 要求 AI 输出问题清单、证据位置、风险等级和建议改法。
5. 人工确认后再改源文件。
6. 重新运行 `check-format`、`check-privacy` 和必要的导出命令。

## Boundaries

AI 可以辅助：

- 发现格式和结构风险。
- 归纳导师或格式意见。
- 草拟修改建议。
- 检查术语、引用和图表编号一致性。

AI 不应替代：

- 学校与学院最新规范判断。
- 导师和作者最终审阅。
- 授权、隐私、数据合规判断。
- 对官方 Word/PDF 成品的人工验收。
