# Official Materials

本目录只记录官方材料的来源和校验信息，默认不提交官方二进制原件。

建议为每份材料记录：

```text
title:
source_url:
access_date:
sha256:
expected_local_path:
redistribution_status: not_committed_by_default
notes:
```

## Policy

- 不默认提交 UCAS 官方 `.doc`、`.docx`、`.pdf`、`.pptx`。
- 不把官方材料散拷到模板、示例或发布包。
- 如果本地导出需要参考 Word 模板，用命令参数传入本机路径。
- 公开发布前逐项确认授权和 checksum。
